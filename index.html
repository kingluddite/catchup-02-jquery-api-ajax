API discussion

  • What does API stand for?
  • API is just a fancy way to say a part of an app that other apps can interact with
  • Jquery Docs we were just looking at are an API "jQuery API documentation"
    • That's because the jQuery API documentation listed all the functions in the jQuery library that our app can use
  • There are lots of different types of APIs
    • As web developers we mostly care about web APIs
    • In this course we'll focus on REST web APIs
  • What is a Web API?
    • A tool for getting information from a database to a user via the web or from a user to a database
  • Think about how Facebook works
    1. You open Facebook in your browser
    2. Your computer sends your user information to the Facebook API
    3. Which in turn gets all your relevant info from the Facebook Database
    4. And sends that info back to your browser
    5. And your browser renders that data so you can see it
  • You sign up for a udemy account
    1. Your computer sends your new user name and password (and other account info) to the Udemy API which permenantly add it to the Database
  • **note** A web API is a mediator of all communication between web app users and Databases
  • Can one application use data from another application? Yes!
    • Airbnb uses the Google Maps data via the Google Maps API
    • Some WordPress blogs show Twitter feeds in the sidebar on the homepage. WordPress is using the Twitter API to do this!
    • When you are searching for cheap flights on kayak it is using APIs from Delta, United, Jet Blue, Spirit and a ton of others to give you that data.
  • Are all APIs free? No. But many are
  • Warning! Reading APIs and understanding them is a skill. It is tough at first but after working on a few dozen differnt APIs you'll get better at this skill. Practice makes perfect
