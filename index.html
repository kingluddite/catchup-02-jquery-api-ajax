Pokemon API

Let's check out an API ourselves

  1. Open Pokeapi Documentation
    • This is the Documentation for the Pokemon API
  2. On the left under Contents (sidebar) click the Pokemon link
    • This will open all the Pokemon endpoints

How do we get information from an API?

By combining the base URL with and endpoint

Click on the logo (the logo image is also a link) and you'll see the base URL

Click the link to go back to the docs and click several links on the Pokemon heading link in the sidebar. Notice how the base url stays the same but the endpoint changes. As that endpoint changes so does the data we get back

Pokemon Colors

  1. Navigate to the Pokemon Colors link under Pokemon in the sidebar
    • This is just the docs
    • But if you grab the base URL and add to it the "pokemon-color/" (this is the link you will get - click to see where it takes you)
    • Notice how it brings up a list of links to information about each Pokemon color group.
    • Click on the link under blue https://pokeapi.co/api/v2/pokemon-color/2/
    • Minimize by clicking to collapse (or scroll) past "names" to pokemon_species
      • See how we have a list of all blue species with links to more information about each one
      • Click one of the species links and see that we have more info on the given species
