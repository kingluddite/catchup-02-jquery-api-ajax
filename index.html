Let's take this Pokemon API as an example

  1. I want to build an app
  2. It is going to be a Fantasy pokemon league online
  3. To do this I would need to start with data on each type of pokemon
  4. I could build this from scratch but that would take a lot of time (and money)
  5. I could use this API data to create a robust app for Pokemon fans without having to independently gather and categorize all Pokemon data
