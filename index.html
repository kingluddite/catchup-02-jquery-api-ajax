AJAX: Star Wars Activity 1

Using the Star Wars API and the starter HTML, complete the following:

Star Wars Page

  1. Generate a random number between 1 and 60.
  2. Use ajax to query the planet for this number (for example if the random number generated was 5), use the query URL https://swapi.co/api/planets/5
  3. log the resulting data to the console.
  4. Append to the root div the name, terrain, and population of the randomly selected planet.

Use the 10 minutes wisely!
